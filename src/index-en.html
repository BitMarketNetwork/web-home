About

The project is a cross-platform multi-currency wallet that allows the user to store various crypto-currencies in one application as conveniently, comfortably and safely as possible, as well as make transfers.

Distinctive features of our wallet:

Comfortable U1/UX

Security of private keys storage and irrelevance of brutforce (with current busting capabilities), including the possibility of including 2-factor authorization

Fast synchronization on all currencies presented

Cross-platform (the first stage is Win/Mac OS X/Linux, in the near future iOS, Android)

Displaying the amount in the fiat for each currency and their total value depending on the chosen site with the rates

No need to register and verify

Open source of the client, which indicates that private keys are not transferred anywhere and are stored securely in the user's local store

The ability to open your wallet e anywhere and at any time by code phrase (a set of words) and get access to the management of your assets

The wallet itself (customer) is a free open source application

No option of impact on the client application by developers
