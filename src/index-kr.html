프로젝트 정보

이 프로젝트는 사용자가 편리하고 편안하고 안전하게 한 응용 프로그램에 다양한 암호화 통화를 저장뿐만 아니라 전송을 할 수 있습니다 크로스 플랫폼 멀티 통화 지갑입니다.

지갑의 독특한 특징 :

편안한 U1 / UX

개인 키 저장 및 2 단계 권한 부여 포함 가능성을 포함하여 brutforce (현재 파열 기능 포함) 의 관련성

제시 된 모든 통화에 대한 빠른 동기화

크로스 플랫폼 (첫 번째 단계는 가까운 미래의 아이폰 OS, 안드로이드에서 승리 / 마스 / 리눅스입니다)

요금으로 선택한 사이트에 따라 각 통화와 총 값에 대한 피아트의 금액을 표시

등록 및 확인 이필요 없음

개인 키가 아무 데도 전송되지 않고 사용자의 로컬 저장소에 안전하게 저장됨을 나타내는 클라이언트의 오픈 소스

코드 문구 (단어 세트) 에 의해 언제 어디서나 지갑 전자를 열고 자산의 관리에 액세스 할 수있는 기능

지갑 자체 (고객)는 무료 오픈 소스 응용 프로그램입니다

개발자가 클라이언트 응용 프로그램에 미치는 영향 의 옵션 없음
